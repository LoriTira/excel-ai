Excel AI

Use =EXCELAI.AI("your prompt") in any cell.

Examples

  • =EXCELAI.AI("What is 2+2?")
  • =EXCELAI.AI("Classify sentiment", A1)
  • =EXCELAI.AI("Summarize", A1:A10)

Troubleshooting
  • Make sure Ollama is running: ollama serve
  • Pull a model: ollama pull llama3.2:1b
  • List available models: ollama list
  • Re-run the install script to fix everything automatically
