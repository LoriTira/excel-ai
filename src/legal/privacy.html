Privacy Policy

Effective date: April 7, 2026

Excel AI is an open-source Microsoft Excel add-in that provides AI-powered custom functions. This policy explains how data is handled when you use the add-in.

Data Collection

Excel AI does not collect, store, or transmit any data to the add-in developer or any third party. There are no analytics, no telemetry, no cookies, and no tracking of any kind.

Data You Send to AI Providers

When you use the =EXCELAI.AI() function, your prompts and any cell data you reference are sent directly to the AI provider you have configured:

  • Local (LM Studio) — data is sent to a server running on your own machine. Nothing leaves your computer.
  • External API — data is sent to the API endpoint you specify (e.g., OpenAI, Anthropic). Your data is subject to that provider's privacy policy.

The add-in acts as a direct client — requests go straight from your browser to the AI provider. No data passes through any intermediary server.

Settings and API Keys

Your configuration (provider choice, server address, API endpoint, API key, and model name) is stored in your browser's localStorage on your device only. This data is never transmitted anywhere.

Caching

AI responses are cached in memory while Excel is open to avoid duplicate requests. This cache is not written to disk and is discarded when you close Excel.

Open Source

Excel AI is open source. You can review the complete source code to verify these practices at github.com/LoriTira/excel-ai.
