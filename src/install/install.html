Install Excel AI

Add =EXCELAI.AI("your prompt") to any Excel cell, powered by local or cloud AI.

macOS
Windows

Quick Install

Open Terminal and paste:

curl -fsSL https://loritira.github.io/excel-ai/install.sh | bash

Uninstall

curl -fsSL https://loritira.github.io/excel-ai/uninstall.sh | bash

Quick Install

Open PowerShell and paste:

irm https://loritira.github.io/excel-ai/install.ps1 | iex

Uninstall

irm https://loritira.github.io/excel-ai/uninstall.ps1 | iex
Manual install (no terminal required)
  1. Download manifest.xml
  2. Open Excel
  3. Go to Insert > My Add-ins > Upload My Add-in
  4. Browse to the downloaded manifest.xml
  5. Click Upload
After installation
  1. Restart Excel (quit and reopen)
  2. Use =EXCELAI.AI("your prompt") in any cell — it works immediately with the local AI model
  3. Click Excel AI in the Home tab to change settings:
    • Local (Ollama) — change the model name (default: qwen2.5:1.5b)
    • External API — use OpenAI, Anthropic, or any compatible API instead
For forks / custom hosting

If you host the add-in at a different URL, pass it to the install script:

curl -fsSL https://your-site.com/install.sh | bash -s -- --url https://your-site.com

On Windows:

irm https://your-site.com/install.ps1 -OutFile install.ps1; .\install.ps1 -BaseUrl https://your-site.com
